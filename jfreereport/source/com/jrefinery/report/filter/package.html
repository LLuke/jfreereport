The filter interface replaces all specialized Elements. The to be deprecated elements are
  • DataElement
  • DateElement
  • DateFunctionElement
  • FunctionElement
  • LabelElement
  • LineShapeElement
  • MultilineTextElement
  • NumberElement
  • NumberFunctionElement
  • StringElement
  • StringFunctionElement

How to use the FilterInterface

All data is read from DataSource implementations. A DataSource encapsulates a single value, which is read by an DataTarget. An object which implements both DataSource and DataTarget acts as filter, it modifies the value before it is returned to the next DataTarget.
