Setup instructions

You will have to manually fill the WEB-INF directory with the following files:

  • WEB-INF/lib/jfreereport-{version}.jar

    JFreeReport with at least version 0.8.1

  • WEB-INF/lib/jfreereport-{version}-demo.jar

    JFreeReport demopackage. The version must fit the jfreereport.jar.

  • WEB-INF/lib/iText-0.96.jar

    An recent iText library. We supply version 0.96.

  • WEB-INF/lib/jcommon-0.7.1.jar

    JCommon must be at least version 0.7.1.

  • WEB-INF/lib/jfreechar-0.9.4.jar

    JFreeChart is used to encode the images for the JPEG-Servlet.

  • WEB-INF/lib/pixie.jar

    Pixie provides support for WMF-Images.

  • WEB-INF/lib/junit.jar

    JUnit is not required here, but needed when rebuilding the project..

  • WEB-INF/lib/bsh-1.2b6.jar

    The BeanShell provides the scripting interface for BSHExpression.

  • WEB-INF/lib/jlfgr-1_0.jar

    This IconCollection is used to fill the tablemodel.

  • WEB-INF/web.xml

    The web.xml defines the available servlets an their mapping into the servlets URI namespace.

    • PDF-Servlet
    • HTML-Servlet
    • First page of the JPEG-Servlet
