Pixie Revision History

Here is the history of changes to Pixie.
    1.13 30th March 1998
    • Renamed packages from UK.co.bhresearch.* to gnu.bhresearch.*.
    • First public release of source code under GNU LGPL.

    1.12 13th December 1997

    • Fixed bug with text from Corel Draw files.
    • Fixed polypolygon bug.
    • Switched to different compiler.

    1.11 5th December 1997

    • Made comma optional in "ConvPxi -r 600,300" so as to work better with DOS batchfiles, which don't pass commas through.
    • Made the progress bar code simpler and faster.

    1.10 1st December 1997

    • Minor changes to error messages and documentation.

    1.09 29th November 1997

    • Fixed bug which made Netscape 3.03 hang.

    1.08 24th November 1997

    • The converter now creates files that are 10% smaller.
    • File format version number changed to 300.
    • Include the number in Pixie file format version error messages.
    • Renamed main convert class from Main to Converter.

    1.07 November 1997

    • Various changes relating to (unpublished) PixieSlide.

    1.06 October 1997

    • First public version of Pixie.
