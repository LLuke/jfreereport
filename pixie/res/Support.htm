Pixie and PixieSlide Feedback and Support

This page explains how to get in touch with the developers of Pixie.

Feedback

    For any further information, to request new features, or just to tell me how you are getting on, please send email to pixie@bhresearch.co.uk.

    If you've used Pixie to make a cool website, let me know. I'd like to keep a gallery of sites for people to look at.

    If you have some ideas for new Pixie features, I'm open to suggestions. No guarantees though.

    If you want to work with the Pixie source code yourself, see the Pixie Source Code section.

Support

    Pixie and PixieSlide are free products, which means there is no money to pay for support. What follows is mostly to help you sort out any problems yourself.

    • First, check for any warning messages that might give you a hint as to what has gone wrong. Look them up in the list of Warning and error messages.

    • Secondly, look at the list of Common Problems to see if yours is listed. (If it isn't, and you think it ought to be, let me know - even if you eventually sort it out yourself.)

    • Thirdly, re-read the documentation. Especially the converter and viewer references, and the PixieScript language reference. Perhaps something which wasn't clear to you the first time through will make more sense now.

    • Fourthly, read through the release history to see if your problem is mentioned there, and to make sure you have the most recent version of the files.

    Finally, if none of that helps, drop me a line at pixie@bhresearch.co.uk. and I'll do what I can.
