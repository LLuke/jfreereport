WARNING: Your browser does not have Java enabled, so you are looking at the JPEG, not the Pixie image. You need to have a Java-capable browser to see the Pixie samples on this and later pages.

Pixie is a small, fast file format for scalable vector images.

You are looking at a Pixie image now, displayed by a Java applet which is only 6k long. The Pixie image itself is less than 1.5k. As a JPEG, because of the radial fill effect, the picture takes 16k. As a GIF it takes 40k.

And it's free. Want to know more?
