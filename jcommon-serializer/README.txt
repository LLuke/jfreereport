1.  INTRODUCTION
-----------------
JCommon is a free general purpose Java class library that is used in
several projects at www.jfree.org, including JFreeChart and
JFreeReport.

This library module contains a general serialization framework. It simplifies
the task of writing custom serialization handlers for non-serializable classes.

JCommon-Serializer is licensed, free of charge, under the terms of the GNU Lesser
General Public Licence.  A copy of the licence is included in the download.

The latest version of this class library can be obtained from:

    http://www.jfree.org/jfreereport/jcommon-serializer

If you have any comments, suggestions or bugs to report, please post a
message in the JFreeReport forum.
